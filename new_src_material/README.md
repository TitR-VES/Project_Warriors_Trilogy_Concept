# new_src_material

This is the light-weight inbox for loose source pages that are ready to sort.

Use this directory when a new source page, insert, or small related set has been
prepared but has not yet been promoted into its durable home.

## Best Fit For This Directory

- a single new character page
- a small mission insert
- a compact manifest plus one or two related source pages
- material that should be sorted soon, not kept indefinitely

## Not A Good Fit

- large multi-file package drops
- frozen historical bundles
- recovered legacy material
- long-term storage for documents that already have a clear home

## Working Rule

Anything that stays here for long should trigger a decision:

- promote it into `dev_src_docs/`
- convert it into a reviewed package in `incoming_patches_and_updates/`
- archive it
- discard it if it was truly temporary
