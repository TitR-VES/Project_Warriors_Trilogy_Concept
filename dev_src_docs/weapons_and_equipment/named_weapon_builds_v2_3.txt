PURPOSE
This registry answers one question consistently across campaigns: which weapon
identities are named builds, what platform do they use, who owns or carries
them, and which source controls the detailed specification.

This file does not duplicate every attachment block. Dedicated build pages and
campaign source packets remain authoritative for detailed configuration.

--------------------------------------------------------------------------------
0. GLOBAL GOVERNANCE CORRECTIONS
--------------------------------------------------------------------------------
1. Stable IDs replace bare ordinal build numbers for cross-project reference.
   Existing BUILD 01–12 labels remain historical aliases only. This resolves
   the WREN / HELL OR HIGH WATER BUILD 08 collision and prevents future drift.
2. PR #22 was still open when this pass was prepared. Any v2.2 wording saying it
   had already merged is superseded. Merge the UI Prompt System dependency
   before or together with this registry pass.
3. DEAD BLOW and DEADBOLT are distinct locked builds. Neither name replaces the
   other.
4. STAPLER is locked canon, not a provisional APC9K candidate.
5. DON'T TREAD retirement is locked for Romeo Game 2, Act II, after the militia
   recognition beat; recognition causes the quiet retirement.
6. GRAY GHOST and NOTHING retain their base builds and link to the completed
   Nightfall aging/configuration source.
7. CATCH-22 the weapon and CATCH-22 the mission/flashback title must be
   disambiguated by context and metadata.
8. LAST RESORT and SAVING GRACE are locked names/platforms with dedicated stubs;
   exact build specifications remain open.
9. EXTREME PREJUDICE is the Riptide Red finale CheyTac carried by CRIMSON THREE,
   Lawrence "Rip" Mansell. It supersedes the older Mk18 finale-primary claim.
