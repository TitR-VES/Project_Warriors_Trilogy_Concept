# Recovered_Sources

This directory holds recovered legacy source extracts.

These files are useful for verification, cross-checking, and reclamation, but
they are not the preferred everyday authoring layer.

## How To Use This Material

- compare recovered text against current source pages
- reclaim valuable details into new stable source pages
- preserve evidence of prior thread or bundle content

## Important Rule

If recovered material becomes live again, promote the reclaimed content into
`dev_src_docs/` as a proper source page instead of continuing to edit the
recovered extract directly.
