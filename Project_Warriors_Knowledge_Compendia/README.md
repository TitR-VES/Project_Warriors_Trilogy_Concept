# Project_Warriors_Knowledge_Compendia

This directory is the normalized working knowledge layer for the repo.

Use these files when you need a compact current view of the project without
opening every underlying source page. They are high-value reference documents,
but they are not a substitute for the durable source pages in `dev_src_docs/`.

## What Lives Here

- the six working compendia
- the archival master
- memory-integration and upload-order guidance

## Editing Rule

- Update compendia after source pages change.
- Avoid doing first-draft authoring only in a compendium when the material
  deserves a dedicated source page.
- Treat the archival master as milestone ballast, not the main day-to-day
  working file.

## Practical Use

If you need fast orientation, start here. If you need to make the project more
correct, make or land the source-page change first.
